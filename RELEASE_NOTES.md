# StarsHub v1.1.0

## 新增
- 设置页改为左侧子导航（通用/AI/嵌入/标签/数据/关于）+ 右侧内容区分页渲染
- 自定义 AI 支持多配置管理：新建/保存/展开编辑/测试连接/启用/删除
- 自定义 AI 配置卡片可展开折叠，展开后可编辑端点/Key/模型
- 嵌入配置连接状态持久显示（已连接/连接失败/测试中）
- 插件启动时自动验证 GitHub Token / 自定义 AI / Embedding 连接
- httpPost 支持 SSL 自签名证书 + gzip/deflate 响应解压

## 修复
- 自定义 API 按钮点击无反应（spread 顺序覆盖 mode 值）
- 自定义 AI 连接 404（端点缺少 /chat/completions 路径自动补全）
- 设置页交互体验差（13 个 card 堆叠改为分组导航）
- 语言设置移除（默认中文）
- 新建配置名称默认用模型名
